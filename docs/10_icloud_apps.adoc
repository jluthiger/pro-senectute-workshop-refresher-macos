= Praktische macOS-Apps mit iCloud-Anbindung
:icons: font
:toc: macro

== Ziele
* Apps wie Notizen, Erinnerungen, Kalender mit iCloud synchronisieren
* Safari-Lesezeichen zwischen Geräten nutzen

== Aufgaben
. Erstellen Sie eine Notiz auf dem Mac und prüfe sie auf einem anderen Gerät.
. Legen eine Erinnerung an und prüfen Sie die Synchronisation.
. Fügen Sie ein Safari-Lesezeichen hinzu und prüfen Sie, ob es synchronisiert wurde.
