= iCloud Fotos & andere iCloud-Dienste
:icons: font
:toc: macro

== Ziele
* iCloud-Fotos einrichten und verstehen
* Weitere iCloud-Dienste (Mail, Notizen, Kalender) nutzen

== Aufgaben
. Aktivieren Sie iCloud-Fotos und beobachte die Synchronisation.
. Loggen Sie sich bei https://www.icloud.com[iCloud] ein und erkunden Sie die Web-Apps.
. Erstellen und synchronisieren Sie eine Notiz oder einen Kalendereintrag.

* Welche iCloud-Dienste sind für dich besonders nützlich?
* Welche Fragen haben Sie zur Datensicherheit?
