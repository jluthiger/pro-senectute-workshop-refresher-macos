== Kursunterlagen

Es lohnt sich im Vorfeld Gedanken über das eigene Setup in Bezug auf die iCloud zu machen. Hierfür steht eine xref:mein-setup.pdf[Vorlage] zur Verfügung, die eigenständig oder gemeinsam mit dem Kursleiter ausgefüllt werden kann. Die Vorlage enthält unter anderem folgende Leitfragen, die bei der Vorbereitung hilfreich sind:

* *Kennen Sie Ihren Cloud-Dienst?* Welcher Anbieter? Welche Funktionen nutzen Sie bereits?

* *Haben Sie Ihren Benutzeraccount?* Zugangsdaten, Postfach, Rechte etc.

* *Kennen Sie die URL des Cloud-Diensts?* Damit Sie sich direkt einloggen können.

Diese Fragen dienen als Orientierung, damit Sie Ihr Setup (Geräte, Verbindungen, Speicherorte usw.) klären und optimieren können. So stellen Sie sicher, dass technische Stolpersteine frühzeitig erkannt und vermieden werden.
