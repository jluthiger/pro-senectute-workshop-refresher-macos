= Screenshots, PDF & ZIP-Dateien
:icons: font
:toc: macro

== Ziele
* Screenshot-Techniken anwenden
* PDFs bearbeiten und kombinieren
* ZIP-Dateien erstellen und entpacken

== Aufgaben
* Screenshots
. Erstellen Sie drei Screenshots (Bildschirm, Auswahl, Fenster).
* PDFs
. Erstellen Sie von jedem Screenshot eine PDF-Datei.
. Bearbeiten Sie eine PDF-Datei in Vorschau.
. Kombinieren Sie zwei PDFs.
* ZIP-Dateien
. Erstellen Sie eine ZIP-Datei aus ihren Screenshots.
. Kopieren und Entpacken Sie die ZIP-Datei in einem anderen Ordner.
