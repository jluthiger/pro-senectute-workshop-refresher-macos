= Finder & Dateiverwaltung
:icons: font
:toc: macro

== Ziele
* Aufbau verstehen, eigene Ablagestruktur erstellen
* Grundprinzipien von Dateiablage vertiefen

== Aufgaben
* Analyse der aktuellen Ablagestruktur
. Zeichnen Sie ihre aktuelle Ordnerstruktur.
. Notieren Sie deren Vor- und Nachteile.
* Planung und Umsetzung einer neuen Struktur
. Entwerfen Sie eine neue, effizientere Ordnerstruktur.
. Erstellen Sie ihre neue Struktur im Finder, z.B. für die Ablage der Dokumente für unterschiedliche Workshops.
. Führen Sie Dateioperationen durch: kopieren, benennen, verschieben, löschen.

== Reflexion
* Wie fühlen Sie mit Ihrer neuen Struktur?
* Werden Sie diese langfristig nutzen? Warum oder warum nicht?
