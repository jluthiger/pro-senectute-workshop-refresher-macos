= Finder & Dateiverwaltung
:icons: font
:toc: macro

== Ziele
* Aufbau verstehen, eigene Ablagestruktur erstellen
* Grundprinzipien von Dateiablage vertiefen

== Aufgaben
. Zeichnen Sie ihre aktuelle Ordnerstruktur und notieren Sie deren Vor- und Nachteile.
. Entwerfen Sie eine neue, effizientere Ordnerstruktur.
. Erstellen Sie ihre neue Struktur im Finder (Ordner anlegen, verschieben, umbenennen).
. Führen Sie Dateioperationen durch: kopieren, verschieben, löschen.

== Reflexion
* Wie fühlen Sie mit Ihrer neuen Struktur?
* Werden Sie diese langfristig nutzen? Warum oder warum nicht?
